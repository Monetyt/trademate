## 一、设计理念

**"专业但不冰冷，简洁但不简陋。"**

TradeMate 的用户是外贸从业者——他们不是设计师，但他们每天用 WhatsApp、Gmail、淘宝。他们对"好看"的直觉已经被这些产品训练过了。我们的设计要让他们觉得"这很专业、很可信"，同时操作路径最短。

## 二、色彩系统

```
主色调
accent          #4F46E5  靛蓝   按钮、选中态、链接
accent-hover    #4338CA  深靛   按钮 hover
accent-light    #EEF2FF  极浅靛  选中背景、高亮区域

中性色
bg              #F8F9FB  页面背景
card            #FFFFFF  卡片背景
text-primary    #1A1D28  主文字
text-secondary  #6B7280  辅助文字
text-tertiary   #9CA3AF  占位符、禁用态
border          #E5E7EB  边框、分割线

语义色
success         #059669  成功/确认
warning         #EA580C  警告
error           #DC2626  错误
```

配色原则：**全站只用一个强调色（靛蓝）**，其余全部中性灰。DeepSeek 的做法——颜色越少越高级。

## 三、字体系统

```css
font-family: -apple-system, BlinkMacSystemFont,
             "Segoe UI", "Noto Sans SC", sans-serif;

/* 字阶 */
Hero      24px / 700 / -0.5px   页面标题
Heading   16px / 600             卡片标题
Body      14px / 400 / 1.6      正文
Caption   13px / 500             标签、按钮
Small     12px / 400             说明文字、时间戳
Tiny      11px / 500 / uppercase 侧边栏分组标题
```

全站只用系统字体——零加载时间，用户看着习惯。

## 五、圆角与阴影

```
圆角：统一 12px（卡片、按钮、输入框）
      小圆角 8px（标签、徽章）

阴影（极克制）：
card:  0 1px 2px rgba(0,0,0,0.04), 0 4px 16px rgba(0,0,0,0.04)
hover: 0 2px 4px rgba(0,0,0,0.06), 0 8px 24px rgba(0,0,0,0.06)

原则：影子只用来区分层级，不是装饰。默认状态影子极浅，
hover 时微微抬升。
```

## 七、移动端布局

```
┌───────────────────┐
│ TradeMate    ⚙️    │  ← 顶部栏（48px）
├───────────────────┤
│ [📧][💬][💰][🔧][🎯]│  ← 横向滑动类型选择器
│  询盘  WA  报价 售后 开发  │
├───────────────────┤
│                   │
│ 粘贴买家询盘       │
│ ┌───────────────┐ │
│ │               │ │  ← 输入框（≥120px 高）
│ │               │ │
│ └───────────────┘ │
│                   │
│ [    生成    ]    │  ← 全宽按钮（≥48px 高）
│                   │
├───────────────────┤
│                   │
│ 生成结果          │
│ ┌───────────────┐ │
│ │               │ │  ← 结果区
│ │               │ │
│ └───────────────┘ │
│ [复制] [分享到WA] │  ← 操作栏（固定底部）
│                   │
├───────────────────┤
│ 🏠 首页 | 📋类型 | 🕐历史 | ⚙️设置 │  ← 底部 TabBar
└───────────────────┘
```

### 移动端特有交互

| 交互 | 说明 |
|------|------|
| 底部 TabBar | 4 个入口：首页（输入）、类型（切换邮件类型）、历史（最近记录）、设置 |
| 横向滑动选择器 | 邮件类型不用下拉菜单，用横向滑动标签，比下拉快 |
| 一键分享 | 生成结果 → "分享到 WhatsApp"，直接唤起 App |
| 长按复制 | 结果区支持长按选择部分文字复制 |
| 下拉刷新 | 首页下拉清空输入框，重新开始 |
| 触控尺寸 | 所有可点击元素 ≥ 44px（iOS HIG 标准） |

## 八、核心组件设计

### 按钮

```
主要按钮（全宽）：bg=accent, text=white, h=48px, radius=12px
次要按钮（内联）：bg=transparent, border=border, h=36px
图标按钮：36×36px, hover 变色
```

### 输入框

```
默认：bg=#FAFBFC, border=border, radius=12px
聚焦：border=accent, bg=white, box-shadow=0 0 0 3px accent/8%
```

### 卡片

```
默认：bg=white, radius=12px, shadow=card
悬停（可点击卡片）：translateY=-1px, shadow=hover
```

### 侧边栏

```
背景：#111827（深灰黑，不是纯黑）
宽度：240px
分组标题：11px uppercase, color=#6B7280
菜单项：13px, color=#D1D5DB, hover 变亮
选中态：左侧 2px accent 色竖线 + 背景 #1F2937
```

### 底部 TabBar（移动端）

```
背景：white, 顶部 0.5px border
高度：56px（含 safe area）
图标 + 文字，选中态 accent 色
4 个 tab 等宽
```

## 九、动效规范

```
transition: all 0.15s ease   —  通用过渡
hover 抬升: transform: translateY(-1px) + shadow 扩散
按钮点击: scale(0.98) → scale(1)（轻微按压感）
结果出现: fadeIn + slideUp, 0.3s
侧边栏切换: 即时（不需要动画，影响操作速度）
加载态: 结果区显示骨架屏或旋转指示器
```
