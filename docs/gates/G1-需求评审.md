## 一、用户画像

### 核心用户

| 画像 | 描述 | 核心场景 |
|------|------|---------|
| 外贸业务员（1-3 年） | 英语能读写但不地道，每天处理 10-30 封买家邮件 + WhatsApp 消息 | 快速回复询盘、报价跟进、开发信 |
| SOHO 外贸创业者 | 一个人做所有事，既是业务员又是跟单又是售后 | 全类型邮件都需要，效率第一 |
| 外贸团队主管 | 管理 3-10 人团队，希望统一回复质量 | 需要模板一致性，看到团队的回复质量 |

### 用户痛点（已验证）

1. 不是"不会写英文"，而是"不确定哪种措辞成交率更高"
2. WhatsApp 沟通占比超 50%，但现有工具只支持邮件
3. 信息散落（ERP/微信/WhatsApp/邮箱），写封邮件要开 5 个地方
4. 凌晨收到消息也要回复——需要随时随地可用
5. 现有竞品（OKKI $50-200/月）太贵，个人买不起

## 二、竞品分析

| 竞品 | 价格 | 优势 | 劣势 |
|------|------|------|------|
| OKKI Leads AI | $50-200/月 | 阿里生态整合，功能全 | 太贵，个人用不起；需要阿里国际站账号 |
| 信风AI | 按结果付费 | 多渠道触达 | 偏销售自动化，不是邮件写作 |
| ChatGPT/Claude 直接使用 | 免费-$20/月 | 灵活 | 没针对外贸优化，每次要写长 Prompt |
| Grammarly | 免费-$12/月 | 语法纠正 | 只会改语法，不会写策略 |
| 我们的 TradeMate | 免费（自付 API） | 10 种外贸邮类型全覆盖，多语言，即开即用 | 品牌认知度零，产品较新 |

### 差异化定位

TradeMate = **外贸专用的 AI 写作助手**，不是通用 AI，不是 CRM，不是自动化营销工具。

核心差异化：每种邮件类型都内置了外贸老手的"策略思维"——不只是帮你写，还告诉你**为什么这样写、什么时机发、怎么跟进**。

## 五、设计基准

参考 DeepSeek 的设计语言：
- 色彩：素雅中性色为主，靛蓝作为唯一强调色
- 留白：内容区大量留白，卡片间距 16-24px
- 阴影：极柔和的阴影 (0 1px 2px + 0 4px 16px, 透明度 < 5%)
- 圆角：统一 12px
- 字体：系统字体栈，标题 24px/700，正文 14px/400
- 图标：Emoji 过渡期，后续升级 SVG 图标
- 动效：hover 微抬 1px + 阴影扩散，过渡 0.15s
- 整体感觉：专业但不冰冷，简洁但不简陋

## 七、移动端策略

### 为什么是 PWA 优先

| 方案 | 开发成本 | 上架 | 更新 | 离线 | 推送 | 适用阶段 |
|------|---------|------|------|------|------|---------|
| PWA | 同一代码库 | 无需审核 | 即时 | ✅ | ✅ | **MVP 首选** |
| React Native | 需重写 UI 层 | App Store + Google Play | 审核 1-3 天 | ✅ | ✅ | 用户量 1000+ |
| 原生 Swift/Kotlin | 两套代码 | 同上 | 同上 | ✅ | ✅ | 用户量 10000+ |

### PWA 能力清单

- 添加到主屏幕（iOS"添加到主屏幕"、Android "安装"）
- 离线缓存：静态资源 + 最近 20 条历史记录
- 推送通知：新邮件类型上线、重要更新
- 原生分享：生成结果 → 一键分享到 WhatsApp/邮件/微信
- 全屏体验：隐藏浏览器地址栏，看起来像原生 App

### 移动端 UI 差异

| 桌面端 | 移动端 |
|--------|--------|
| 左侧深色导航栏（240px 固定） | 底部 TabBar（4 个主入口） |
| 弹窗设置 | 全屏设置页 |
| 侧边栏切换邮件类型 | 顶部横向滑动分类 + 下拉选择 |
| 结果区旁边工具栏 | 结果区底部固定操作栏 |
